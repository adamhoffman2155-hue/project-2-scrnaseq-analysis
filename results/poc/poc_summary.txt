PROOF OF CONCEPT - scRNA-seq workflow on PBMC3k
============================================================

Scope note:
  This POC validates the scRNA-seq infrastructure only.
  It does NOT answer the MSI / tumor-immune question.
  The full MSI-GEA analysis needs a larger dataset
  (e.g. Pelka 2021 CRC atlas, ~2 GB) and is out of
  scope for a 1-day POC.

Dataset: 10x Genomics PBMC 3k
  Upstream: https://cf.10xgenomics.com/samples/cell-exp/1.1.0/pbmc3k/pbmc3k_filtered_gene_bc_matrices.tar.gz
  Loader: sc.datasets.pbmc3k() with fallback to raw 10x mtx files (barcodes/genes/matrix) mirrored on GitHub.

Caveats:
  - PBMC3k is a benchmarking dataset, not a tumor sample.
  - No MSI / tumor metadata available here.
  - Marker-based annotation uses mean expression of a
    small canonical gene set; borderline clusters may
    collapse onto the dominant type (e.g. CD4/CD8 T merge).
  - The canonical PBMC3k has 8 clusters including a
    tiny megakaryocyte (PF4/PPBP) cluster and a
    dendritic (FCER1A/CLEC10A) cluster. With this marker
    set (no MK, small DC set) those clusters are
    mis-assigned by argmax — e.g. in one run cluster 3
    (SDPR/PF4/GNG11/PPBP markers = megakaryocyte) was
    labelled Monocyte. Inspect rank_genes_groups output
    before trusting the label column for small clusters.
